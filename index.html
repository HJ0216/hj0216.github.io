자기 소개

업무의 의미와 가치를 고민합니다.

단순하게 구현 가능 여부보다, 사용자 가치와 함께 개발 효율성까지 고려하고자 합니다.

기존 방식보다 효율적인 방법이 있다면, 근거를 들어 설명하고 대안을 제시하고자 합니다.


책임감있게 행동합니다.

첫 실무 프로젝트에서 단독으로 프로젝트를 맡아 끝까지 책임지고 성공적으로 마무리하였습니다.

부족한 자료는 API 문서뿐만 아니라 GitHub에 공개된 다양한 코드를 참고하였습니다.

또한, 복잡한 문제는 동료와 논의하고, API 포럼에도 질문을 남기며 끝까지 해결하고자 하였습니다.


성실함을 증명할 수 있습니다.

2년 이상 블로그와 GitHub를 통해 배운 내용을 정리하고 공유하고 있습니다.

업무와 개인 공부에서 얻은 내용을 각각 TIL과 게시글로 남기며 꾸준히 기록하고 있습니다.


스스로에게 맞는 효율적인 업무 흐름을 고민하며, 저만의 작업 방식을 만들어나가는 것을 좋아합니다.

프로젝트 초기에 핵심 기능 구현에 집중하고, 이후 테스트와 리팩토링에 시간을 투자하여 프로그램의 완성도를 높일 수 있었습니다.

이러한 경험을 바탕으로, 일정 안에서 완성도 있는 결과물을 만드는 믿음직한 개발자가 되고자 합니다.

기술

Java

Java, Spring Boot, JPA를 사용하여 MVC 패턴 기반의 프로젝트를 진행한 경험이 있습니다.

이전 프로젝트를 리팩토링하며, REST API, JWT, Test Code 등을 추가적으로 학습하였습니다.

현재 Spring Boot 관련 프로젝트 강의를 수강하고 있습니다.


C#

WPF(C#, XAML)를 사용하여 윈도우 응용프로그램을 개발하고 있습니다.

동기 및 비동기 처리를 통해 UI 응답성을 개선할 수 있습니다.

SQL과 LINQ를 활용하여 데이터 처리 속도를 개선한 경험이 있습니다.


Database

테이블을 설계하며 키와 관계 설정, 정규화를 통해 데이터의 무결성을 높이기 위해 고민해 본 경험이 있습니다.

Trigger를 사용하여 데이터베이스 작업을 자동화해 본 경험이 있습니다.

Redis를 활용한 캐싱 전략을 통해 데이터베이스 부하를 줄이고, 응답 속도를 개선하는 방법을 학습하였습니다.


Cloud

S3, EC2, RDS 등 AWS의 기본 서비스를 활용한 클라우드 기초를 학습하였습니다.

Auto Scaling과 Load Balancer 실습을 통해 안정적인 서비스 운영 방식에 대해 학습하였습니다.
